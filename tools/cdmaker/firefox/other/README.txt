Installing Firefox 3.6
***************************************************************************

Windows 2000/XP/Vista/7:
  1. Insert the CD.
  2. Grant permission to Auto Play and to UAC (User Account Control) if asked.
  3. Click on "Run FirefoxSetup3.6.exe" to start the installer.
  4. Follow the prompts and instructions in the installer.
  5. Double click on the "Mozilla Firefox" icon the installer creates on your desktop
     and start browsing.

Mac OS X:
  1. Insert the CD.
  2. Open the CD icon on your desktop.
  3. Open the "Mac OS X" folder.
  4. Open the Firefox 3.6 disk image (Firefox 3.6.dmg) or if you would like a locale other
     than en-US, go into the subfolder for that locale and open the Firefox 3.6.dmg for that locale.
  5. Drag the "Firefox" icon to your Applications folder.
  6. Open your Applications folder and double click "Firefox" to start
     browsing.

Linux:
  1. Open the "Linux" folder.
  2. If you would like to install a different locale from the Linux folder other than en-US
     go into the directory for that locale.
  3. Copy the archive file (firefox-3.6.tar.bz2) to the folder in which
     you wish to install Firefox.
  4. Extract Firefox from the archive using the command 
     "tar -xzvf firefox-3.6.tar.bz2"
     
***************************************************************************
Installing a specific locale of Firefox 3.6 on Windows
***************************************************************************

This CD contains the following locales:
"de", "fr", "es-ES", "ru", "en-GB", "pt-BR", "pl", "it", "ja", and "en-US"

To install a particular locale on Windows:
  1. Insert the CD.
  2. Select Start, then Computer (or My Computer for older versions of Windows).
  3. Right-Click on the Firefox CD and select Open or Explore to browse the contents of the disk.
  4. Open the Windows folder to see the locale folders, select the locale you wish to install and
double-click on the installer to start the wizard. 
  5. Follow the prompts and instructions in the installer
  6. Double click on the "Mozilla Firefox" icon the installer creates on your desktop
     and start browsing.
