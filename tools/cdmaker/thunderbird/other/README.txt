System Requirements
***************************************************************************

Windows
Minimum System Requirements

    * 233 MHz processor
    * 64 MB of RAM
    * 52 MB of free disk space
    * Microsoft Windows 98

Mozilla Recommends

    * 500 MHz processor
    * 256 MB of RAM
    * Microsoft Windows XP

Mac
Minimum System Requirements

    * PowerPC G3 processor
    * 128 MB of RAM
    * 200 MB of free disk space
    * Mac OS X 10.2.x

Mozilla Recommends

    * PowerPC G4 or Intel processor
    * 512 MB of RAM

Linux
Minimum System Requirements

    * 233 MHz processor
    * 64 MB of RAM
    * 52 MB of free disk space
    * Linux kernel 2.2.14 (with glibc 2.3.2, XFree86-3.3.6, gtk+2.0, 
      fontconfig/xft and libstdc++5)

Mozilla Recommends

    * 500 MHz processor
    * 256 MB of RAM
    * a modern Linux distribution

Installation Instructions
***************************************************************************

Installing Thunderbird 2

Windows:
  1. Insert the CD, the Thunderbird installer will start automatically.
  2. Follow the prompts and instructions in the installer.
  3. Double click on the "Mozilla Thunderbird" icon on your desktop.

Mac OS X:
  1. Insert the CD.
  2. Open the CD icon on your desktop.
  3. Open the "Mac OS X" folder.
  4. Open the Thunderbird 2 disk image (Thunderbird 2.0.0.0.dmg).
  5. Drag the "Thunderbird" icon to your Applications folder.
  6. Open your Applications folder and double click "Thunderbird".

Linux:
  1. Open the "Linux" folder.
  2. Copy the archive file (thunderbird-2.0.0.0.tar.gz) to the folder in which
     you wish to install Thunderbird.
  3. Extract Thunderbird from the archive using the command 
     "tar -xzvf thunderbird-2.0.0.0.tar.gz"
